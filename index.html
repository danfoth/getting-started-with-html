Basics of the Web

The web is a collection of HTML (HyperText Markup Language) documents.
These pages are stored on servers and accessed through your browser sending an HTTP request
to the server.

Browser

The browser is a program that enables the user to view web pages. Internet Explorer,
Google Chrome, and Firefox are all examples of browsers.

Servers

Servers are computers with no real frontend. Their purpose is merely to host the HTML
web document.

HTML

HTML mainly contains two elements; text and markup. Text is the general text content
and markup is the style of the text.

Tags

Tags provide the ability to add different styles to the text. A good list of these tags can be
found here. The combination of the opening and closing tags, along with the content is
known as an element.
